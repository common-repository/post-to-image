== Description ==
Post To Image - easy and fast way to generate JPG pictures from your posts.

= Key Features =
*	Simple & fast
*	No JS required, no CSS inserted
*	The image for each post is generated once, when url is open, on the fly
*	When you re-open the URL, the image is not generated, but is opened from the file system
*	Just use shortode [post2image]

= Demo =
[Click here](https://essayzz.com/essays/hyperloop-transportation-system/) and look at the right sidebar

= Languages =
Instruction page in admin panel available only in English.
Post in any language can be converted to the image :)

== Installation ==
1. Upload the `post2image` folder to the `/wp-content/plugins/` directory
2. Activate the "Post To Image" plugin through the WordPress 'Plugins' dashboard.
3. Add to any page using shortcode `[post2image]`

== Frequently Asked Questions ==
= How I can use Post To Image? =
Now you can use Post To Image only by shortcode: add to any page or widget [post2image]
